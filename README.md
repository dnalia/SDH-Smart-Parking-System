# Arduino IoT Project – Button, Buzzer, Servo & I2C LCD

# Description
Developed an Arduino-based IoT system integrating input/output control, LCD display, and servo motor. Demonstrated basic programming concepts, control structures, and hardware interfacing.

# Key Features
- Button & Buzzer: Turns buzzer on/off when button pressed.
- I2C LCD Display: Shows messages and dynamic data.
- Servo Motor Control: Potentiometer input controls servo, with real-time LCD display.
- Setup: Arduino IDE configured with correct board (Arduino Uno) and COM port.
- Libraries Used: Wire, LiquidCrystal_I2C, Servo

# Tech Stack
Arduino, C++, Blynk IoT, I2C LCD, Servo Motor
